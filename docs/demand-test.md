# 페이븐 수요 테스트 플레이북 (2026-06-26 작성, 2주 실행)

> **목적:** 앱스토어·마케팅에 돈/시간 쓰기 *전에*, "진짜 쓸 사람·올 사람이 있나"를 싸게 확인.
> 경쟁 재평가 결론 = 제품 차별점은 얇고(레드오션) **병목은 유통**. 사장님 네트워크 = 콜드(온라인)뿐.
> → **제품(트랙 A)** 과 **유통(트랙 B)** 을 따로 검증한다. **B가 더 결정적.**

## 가설 (이걸 검증)
"사람이 정산을 하나?"(당연) 가 아니라:
- **A(제품):** 실제 모임에서 *복잡한* 정산이 생겼을 때, payven의 경험이 한 팀이라도 "오 이거 편하다" 하게 만드나? 그 중 한 명이 *다음엔 자기가* 정산을 만드나?(viewer→creator)
- **B(유통):** payven을 모르는 사람이 (검색/커뮤니티로) **흘러들어와 실제로 정산을 만드나?**

## GO / KILL (테스트 *전에* 못 박기)
**트랙 A — 제품**
- 🟢 GO: 참여자 중 viewer→creator 1건 이상 · "안 마신 술값 빼는 거/메뉴별/영수증 스캔" 같은 **복잡 가치에 자발적 반응** · 총무가 안 시켰는데 또 씀
- 🔴 KILL: "어 그냥 카뱅으로 하지" 미지근 · 시켜서 한 번 쓰고 끝 · 결국 그냥 1/N · 단계 많아 이탈

**트랙 B — 유통 (더 결정적)**
- 🟢 GO: 커뮤니티 글에 진짜 "써볼게요" 반응 + **Analytics에 모르는 사람이 만들기까지 도달**(바운스 아님)
- 🔴 KILL: 글에 무반응/"카뱅 쓰면 되지" · 랜딩 와도 만들기 0 · 검색량 거의 0(SEO 채널 없음)

🟡 애매 → 테스트 연장 or 포지셔닝 더 좁히기.

---

## 트랙 A — 실제 1건 (2주 내 그 자리)
사장님이 총무일 가능성 높음. **그 자리에 진짜로 payven을 쓴다.**

**원칙: 절대 과하게 가르치지 말 것.** 링크만 주고 "이걸로 정산해봐" 한 줄. 헤매는 지점이 곧 데이터.

**관찰 체크리스트 (총무 첫 사용):**
- [ ] 로그인 게이트에서 막히나?(만들기=로그인 필요 = 첫 이탈 포인트)
- [ ] 멤버·금액 입력 헤매나? 몇 분?
- [ ] **복잡 정산(항목별·차등·안 마신 사람 빼기)을 실제로 쓰나, 그냥 1/N 하나?** ← 핵심
- [ ] 영수증 스캔(OCR) 시도하나? 반응?
- [ ] 결과 링크를 진짜 단톡방에 공유하나?

**관찰 (링크 받은 참여자):**
- [ ] 자기 몫 바로 이해? · [ ] 토스 송금 버튼 누르나? · [ ] "이거 뭐야 편하다" 자발적 반응? · [ ] **누가 자기 정산을 새로 만드나?**(최강 신호)

**디브리핑 (끝나고, 비유도 3개만):**
1. "다음에도 쓸 것 같아? (아니면 왜?)"
2. "이거 없었으면 뭘로 했을 것 같아?" (카뱅/토스/머릿속 → 진짜 경쟁자 확인)
3. "어디가 제일 불편했어?"

## 트랙 B — 콜드 유통 probe (모르는 사람도 오나)
네트워크 없으니 이게 진짜 시험대. **돈 안 쓰고** 한 줌이라도 데려와 본다.

### B-1. 커뮤니티 시딩 (사장님, 직접 글)
**훅 = 또 다른 더치페이 앱(X). 차별 페인으로 시작:**
> "영수증 찍으면 메뉴 자동입력 + 안 먹은 거 빼고 + 무가입 링크로 각자 확인하는 정산기 만들었어요."

**채널 (네트워크 없는 1인 개발자에 맞는 순서):**
- ★ **디스콰이엇 disquiet.io** — 한국 메이커 커뮤니티. "만들었어요" 글 환영·정직한 초기 유저·홍보 금지 아님. **1순위.**
- ★ **GeekNews news.hada.io** — 한국형 HN. 기술 얼리어답터. "Show GN" 류로 공유.
- **본인 SNS(스레드/X/인스타)** — "마땅한 데 없다" 했지만 본인 계정은 콜드 채널 하나. "이런 거 만들었어요" 1글.
- **오픈카톡 정산/더치페이방** — 실 페인 현장이나 홍보 금지 많음 → 질문답변형으로만.
- **네이버 카페(여행·동호회)·에브리타임** — 대부분 홍보 금지 → "더치페이 어떻게 나눠요" 질문에 답하며 자연 노출.

⚠️ 대부분 커뮤니티는 노골 홍보 차단 → **메이커 스팟(디스콰이엇/긱뉴스)은 "피드백 주세요", 일반 커뮤니티는 "기존 질문에 답하며 도구 소개"** 로 접근.

### B-2. SEO 키워드 실검색량 (사장님, 30분)
[네이버 검색광고](https://searchad.naver.com) → 도구 → 키워드 도구. 월간검색수 확인:
`더치페이 계산기` · `더치페이` · `더치페이 앱` · `모임 정산` · `회비 정산` · `여행 경비 정산` · `회식비 정산` · `엔빵 계산기` · `술값 계산기` · `1/n 계산기` · `정산 앱`
- 합쳐 **월 수천~만+** → 검색 유통 채널 살아있음(SEO 랜딩 투자 근거)
- 거의 **0** → 검색으론 못 먹음 = 입소문/커뮤니티만 남음(유통 더 좁아짐)

### B-3. Web Analytics (코드, ~30분 — 콜드 유입 읽는 눈)
`@vercel/analytics` 설치 + 대시보드 ON. 모르는 사람은 관찰 못 하니 **숫자로** 봐야 함:
방문 → 만들기 시작 → 정산 생성 → 링크 공유 → 재방문 퍼널. (개인정보처리방침에 분석 도구 한 줄 고지 권장.)

## 2주 일정 (예시)
- D0~D2: Analytics 켜기(B-3) · 키워드 검색량(B-2) · 커뮤니티 글 1~2개(B-1)
- D3~D10: 실제 자리에 payven 사용(A) + 관찰·디브리핑 · 커뮤니티 반응 모니터
- D11~D14: Analytics 숫자 정리 → **GO / KILL / 피벗 결정 회의(혼자라도 글로)**

## 결정
- **GO** (앱스토어·SEO 투자 진행): A에서 viewer→creator + 복잡 가치 반응 **그리고** B에서 콜드 유입이 0이 아님.
- **KILL/피벗**: 둘 다 미지근 → 취미 출시(이미 라이브니 유지)하되 성장 투자 중단, or 포지셔닝/타깃 재정의.
- 한쪽만 통과 = 🟡 → 통과한 쪽으로 좁혀 재시도.
